-> Project Overview
This repository contains Version 2 of the client website redesign, built using the 'Google AI Stack' (Gemini). 
The goal was to create a modern, mobile-responsive, and static website for "Novel Idea," a local independent bookstore.

-> Tech Stack
HTML5: S emantic structure for better accessibility and SEO.
CSS3:  Custom properties (variables), Flexbox, and CSS Grid for layout.
No JavaScript: The site relies purely on CSS for responsiveness and the mobile menu to ensure speed and stability.

-> How to Run
1.  Clone this repository.
2.  Navigate to the root folder `client-v2-sailor`.
3.  Open `index.html` in any web browser (Chrome, Edge, Firefox).
4.  Navigate through the links (Home, Catalog, Events, Community, About, Contact).

-> AI Workflow (Google Stack)
   Primary Tool: Google Gemini and Google Antigravity.
   Usage: Used to generate the initial file structure, CSS color palette, and responsive grid layouts.
